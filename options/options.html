Configure your integrations

GitHub

Create a token at github.com/settings/tokens/new

Required scope: repo (Full control of private repositories)

This is the first checkbox in the list. It allows reading commits from private organization repositories.

Your GitHub username for filtering commits

Organization name to search repositories in

Azure DevOps

Create a token at dev.azure.com/_usersSettings/tokens

Settings:

  • Organization: Select your organization or "All accessible organizations"
  • Expiration: Set as needed (max 1 year)
  • Scopes: Select "Custom defined"
  • Work Items: Check Read

Your Azure DevOps organization name (from URL: dev.azure.com/organization)
