Reporting Commits Tracker

Version 1.0.2

Features

Commits Tracking

Automatically fetch and display your commits from GitHub and Azure DevOps for any selected date. Perfect for daily reporting.

  • Support for GitHub organizations
  • Support for Azure DevOps projects
  • Filter by source (GitHub, DevOps, or both)
  • Click to copy commit messages

DevOps Tasks

View all work items assigned to you in Azure DevOps.

  • Filter tasks by project
  • See task type (Bug, Task, User Story, etc.)
  • Click to copy task ID
  • Quick link to open in DevOps

Ticket Mappings

Create mappings between keywords (slugs) and ticket numbers for automatic ticket detection in commit messages.

  • Map multiple tickets to one slug
  • Auto-suggest tickets from DevOps tasks
  • Edit and delete mappings

Integration with rep.smartcloud.com.ua

When you select a date on the reporting page, the extension automatically loads commits for that day.

  • Automatic date detection
  • Icon turns green when commits found
  • Quick access via popup

Quick Start

  1. Open extension settings and configure your GitHub and/or Azure DevOps credentials
  2. Select which source to use for commits (GitHub, DevOps, or both)
  3. Go to rep.smartcloud.com.ua and click on a reporting day
  4. Click the extension icon to see your commits
  5. Click on a commit to copy its message

Keyboard Shortcuts

  • Click on commit - Copy commit message
  • Click on ticket badge - Copy ticket number
  • Click on task ID - Copy task ID
